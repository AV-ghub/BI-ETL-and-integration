### **1. –ü—Ä–æ–¥–≤–∏–Ω—É—Ç–æ–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏—è Derived Column**

- **–ù–∞–∑–Ω–∞—á–µ–Ω–∏–µ:** –†–µ–∞–ª–∏–∑–∞—Ü–∏—è **—Å–ª–æ–∂–Ω–æ–π —Ç–µ–∫—Å—Ç–æ–≤–æ–π –ª–æ–≥–∏–∫–∏**, –∫–æ—Ç–æ—Ä—É—é –Ω–µ–ª—å–∑—è –≤—ã–ø–æ–ª–Ω–∏—Ç—å –ø—Ä–æ—Å—Ç—ã–º–∏ —Å—Ç—Ä–æ–∫–æ–≤—ã–º–∏ —Ñ—É–Ω–∫—Ü–∏—è–º–∏.

    *   **–†–µ—à–µ–Ω–∏–µ —Å –ø–æ–º–æ—â—å—é –≤—ã—Ä–∞–∂–µ–Ω–∏–π –≤ Derived Column:**

        *   –°–æ–∑–¥–∞—ë—Ç—Å—è —Å–ª–æ–∂–Ω–æ–µ –≤—ã—Ä–∞–∂–µ–Ω–∏–µ, –∫–æ—Ç–æ—Ä–æ–µ –∏—â–µ—Ç –ø—Ä–æ–±–µ–ª—ã –∏ —Ç–æ—á–∫–∏, –∞–Ω–∞–ª–∏–∑–∏—Ä—É–µ—Ç –¥–ª–∏–Ω—É –∏ –∏–∑–≤–ª–µ–∫–∞–µ—Ç —á–∞—Å—Ç–∏ —Å—Ç—Ä–æ–∫–∏.

### **2. –ü—Ä–æ–¥–≤–∏–Ω—É—Ç—ã–π Fuzzy Lookup –∏ Fuzzy Grouping (Enterprise Edition)**

–≠—Ç–æ **—Å–∞–º–∞—è —Å–∏–ª—å–Ω–∞—è —Å—Ç–æ—Ä–æ–Ω–∞ SSIS** –≤ –æ–±–ª–∞—Å—Ç–∏ –æ—á–∏—Å—Ç–∫–∏. –ü–æ–∑–≤–æ–ª—è–µ—Ç –Ω–∞—Ö–æ–¥–∏—Ç—å —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–∏—è –∏ –¥—É–±–ª–∏–∫–∞—Ç—ã **–¥–∞–∂–µ –ø—Ä–∏ –Ω–∞–ª–∏—á–∏–∏ –æ–ø–µ—á–∞—Ç–æ–∫, —Ä–∞–∑–Ω—ã—Ö —Ñ–æ—Ä–º–∞—Ç–æ–≤ –Ω–∞–ø–∏—Å–∞–Ω–∏—è –∏ –∞–±–±—Ä–µ–≤–∏–∞—Ç—É—Ä**.

#### **A. Fuzzy Lookup (–ù–µ—á—ë—Ç–∫–∏–π –ø–æ–∏—Å–∫)**
- **–ù–∞–∑–Ω–∞—á–µ–Ω–∏–µ:** –°–æ–ø–æ—Å—Ç–∞–≤–∏—Ç—å "–≥—Ä—è–∑–Ω—ã–µ" –¥–∞–Ω–Ω—ã–µ –∏–∑ –æ—Å–Ω–æ–≤–Ω–æ–≥–æ –ø–æ—Ç–æ–∫–∞ —Å "—á–∏—Å—Ç—ã–º" —Å–ø—Ä–∞–≤–æ—á–Ω–∏–∫–æ–º, –Ω–∞—Ö–æ–¥—è **–Ω–∞–∏–±–æ–ª–µ–µ –≤–µ—Ä–æ—è—Ç–Ω—ã–µ —Å–æ–≤–ø–∞–¥–µ–Ω–∏—è**, –¥–∞–∂–µ –µ—Å–ª–∏ —Å—Ç—Ä–æ–∫–∏ –Ω–µ –∏–¥–µ–Ω—Ç–∏—á–Ω—ã.
- **–ö–∞–∫ —Ä–∞–±–æ—Ç–∞–µ—Ç:** –ò—Å–ø–æ–ª—å–∑—É–µ—Ç –∞–ª–≥–æ—Ä–∏—Ç–º—ã, –æ—Å–Ω–æ–≤–∞–Ω–Ω—ã–µ –Ω–∞ **—Ä–∞—Å—Å—Ç–æ—è–Ω–∏–∏ –õ–µ–≤–µ–Ω—à—Ç–µ–π–Ω–∞** (–∫–æ–ª–∏—á–µ—Å—Ç–≤–æ edits: –≤—Å—Ç–∞–≤–æ–∫, —É–¥–∞–ª–µ–Ω–∏–π, –∑–∞–º–µ–Ω —Å–∏–º–≤–æ–ª–æ–≤). –í—ã—á–∏—Å–ª—è–µ—Ç **–∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç —Å—Ö–æ–¥—Å—Ç–≤–∞ (Similarity Score)** –æ—Ç 0 –¥–æ 1.
- **–ö–æ–Ω—Ñ–∏–≥—É—Ä–∞—Ü–∏—è (–≤–∞–∂–Ω–µ–π—à–∏–µ –ø–∞—Ä–∞–º–µ—Ç—Ä—ã):**
    *   **Reference Table:** "–≠—Ç–∞–ª–æ–Ω–Ω–∞—è" —á–∏—Å—Ç–∞—è —Ç–∞–±–ª–∏—Ü–∞ (—Å–ø—Ä–∞–≤–æ—á–Ω–∏–∫).
    *   **Similarity Threshold (–ü–æ—Ä–æ–≥ —Å—Ö–æ–¥—Å—Ç–≤–∞):** –ú–∏–Ω–∏–º–∞–ª—å–Ω—ã–π –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç (–Ω–∞–ø—Ä–∏–º–µ—Ä, 0.8), –ø—Ä–∏ –∫–æ—Ç–æ—Ä–æ–º —Å–æ–≤–ø–∞–¥–µ–Ω–∏–µ —Å—á–∏—Ç–∞–µ—Ç—Å—è —É—Å–ø–µ—à–Ω—ã–º. –ß–µ–º –≤—ã—à–µ –ø–æ—Ä–æ–≥, —Ç–µ–º —Å—Ç—Ä–æ–∂–µ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–∏–µ.
    *   **Token Delimiters:** –°–∏–º–≤–æ–ª—ã, –ø–æ –∫–æ—Ç–æ—Ä—ã–º —Å—Ç—Ä–æ–∫–∞ —Ä–∞–∑–±–∏–≤–∞–µ—Ç—Å—è –Ω–∞ —Ç–æ–∫–µ–Ω—ã (—Å–ª–æ–≤–∞) –¥–ª—è —Å—Ä–∞–≤–Ω–µ–Ω–∏—è (–ø—Ä–æ–±–µ–ª, —Ç–æ—á–∫–∞, –∑–∞–ø—è—Ç–∞—è).
- **–ü—Ä–∏–º–µ—Ä –∏–∑ –∫–Ω–∏–≥–∏:** –ü–æ–¥—Å—Ç–∞–Ω–æ–≤–∫–∞ –ø—Ä–∞–≤–∏–ª—å–Ω–æ–≥–æ –Ω–∞–∑–≤–∞–Ω–∏—è –∫–æ–º–ø–∞–Ω–∏–∏.
    *   **–í—Ö–æ–¥–Ω–æ–π –ø–æ—Ç–æ–∫:** "Micosoft Corp", "Microsoft", "MSFT".
    *   **–°–ø—Ä–∞–≤–æ—á–Ω–∏–∫:** "Microsoft Corporation".
    *   **–†–µ–∑—É–ª—å—Ç–∞—Ç Fuzzy Lookup:** –í—Å–µ —Ç—Ä–∏ –≤–∞—Ä–∏–∞–Ω—Ç–∞ –±—É–¥—É—Ç —Å–æ–ø–æ—Å—Ç–∞–≤–ª–µ–Ω—ã —Å —ç—Ç–∞–ª–æ–Ω–æ–º —Å —Ä–∞–∑–Ω—ã–º–∏ –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç–∞–º–∏ —Å—Ö–æ–¥—Å—Ç–≤–∞ (–Ω–∞–ø—Ä–∏–º–µ—Ä, 0.93, 0.95, 0.65). –ú–æ–∂–Ω–æ –Ω–∞—Å—Ç—Ä–æ–∏—Ç—å –ø–æ—Ä–æ–≥ —Ç–∞–∫, —á—Ç–æ–±—ã "MSFT" –Ω–µ —Å—á–∏—Ç–∞–ª—Å—è —Å–æ–≤–ø–∞–¥–µ–Ω–∏–µ–º.
- **–í—ã—Ö–æ–¥—ã:** –ü–æ—Ç–æ–∫ —Å –Ω–∞–π–¥–µ–Ω–Ω—ã–º–∏ —Å–æ–≤–ø–∞–¥–µ–Ω–∏—è–º–∏ (–∏ –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç–æ–º) –∏ –ø–æ—Ç–æ–∫ –±–µ–∑ —Å–æ–≤–ø–∞–¥–µ–Ω–∏–π.

**–î–ª—è —Å–æ–±–µ—Å–µ–¥–æ–≤–∞–Ω–∏—è:** *"Fuzzy Lookup –∏—Å–ø–æ–ª—å–∑—É—é, –∫–æ–≥–¥–∞ –¥–∞–Ω–Ω—ã–µ –∏–∑ –∏—Å—Ç–æ—á–Ω–∏–∫–∞ —Å–æ–¥–µ—Ä–∂–∞—Ç –æ–ø–µ—á–∞—Ç–∫–∏ –∏–ª–∏ –≤–∞—Ä–∏–∞—Ü–∏–∏ –Ω–∞–ø–∏—Å–∞–Ω–∏—è, –∞ —É –º–µ–Ω—è –µ—Å—Ç—å —ç—Ç–∞–ª–æ–Ω–Ω—ã–π —Å–ø—Ä–∞–≤–æ—á–Ω–∏–∫. –ù–∞–ø—Ä–∏–º–µ—Ä, –¥–ª—è –Ω–æ—Ä–º–∞–ª–∏–∑–∞—Ü–∏–∏ –Ω–∞–∑–≤–∞–Ω–∏–π –≥–æ—Ä–æ–¥–æ–≤ –∏–ª–∏ –∫–æ–º–ø–∞–Ω–∏–π –∏–∑ —Ä–∞–∑–Ω—ã—Ö —Å–∏—Å—Ç–µ–º."*

#### **B. Fuzzy Grouping (–ù–µ—á—ë—Ç–∫–∞—è –≥—Ä—É–ø–ø–∏—Ä–æ–≤–∫–∞)**
- **–ù–∞–∑–Ω–∞—á–µ–Ω–∏–µ:** **–ù–∞–π—Ç–∏ –¥—É–±–ª–∏–∫–∞—Ç—ã *–≤–Ω—É—Ç—Ä–∏ —Å–∞–º–æ–≥–æ –Ω–∞–±–æ—Ä–∞ –¥–∞–Ω–Ω—ã—Ö*** (–±–µ–∑ —ç—Ç–∞–ª–æ–Ω–Ω–æ–≥–æ —Å–ø—Ä–∞–≤–æ—á–Ω–∏–∫–∞). –°–≥—Ä—É–ø–ø–∏—Ä–æ–≤–∞—Ç—å –ø–æ—Ö–æ–∂–∏–µ –∑–∞–ø–∏—Å–∏, —á—Ç–æ–±—ã –æ—Å—Ç–∞–≤–∏—Ç—å –æ–¥–Ω—É –∫–∞–Ω–æ–Ω–∏—á–µ—Å–∫—É—é.
- **–ö–∞–∫ —Ä–∞–±–æ—Ç–∞–µ—Ç:** –°–∞–º–æ—Å—Ç–æ—è—Ç–µ–ª—å–Ω–æ –∞–Ω–∞–ª–∏–∑–∏—Ä—É–µ—Ç –≤—Ö–æ–¥–Ω–æ–π –ø–æ—Ç–æ–∫, –Ω–∞—Ö–æ–¥–∏—Ç –∫–ª–∞—Å—Ç–µ—Ä—ã –ø–æ—Ö–æ–∂–∏—Ö —Å—Ç—Ä–æ–∫ –∏ –¥–ª—è –∫–∞–∂–¥–æ–π –≥—Ä—É–ø–ø—ã:
    *   –í—ã–¥–µ–ª—è–µ—Ç **–∫–∞–Ω–æ–Ω–∏—á–µ—Å–∫—É—é –∑–∞–ø–∏—Å—å (canonical)** (–Ω–∞–∏–±–æ–ª–µ–µ –≤–µ—Ä–æ—è—Ç–Ω–æ –ø—Ä–∞–≤–∏–ª—å–Ω—É—é).
    *   –ü—Ä–∏—Å–≤–∞–∏–≤–∞–µ—Ç –∫–∞–∂–¥–æ–π –∏—Å—Ö–æ–¥–Ω–æ–π —Å—Ç—Ä–æ–∫–µ **–∫–ª—é—á –≥—Ä—É–ø–ø—ã (`_key_in`)** –∏ **–∫–ª—é—á –∫–∞–Ω–æ–Ω–∏—á–µ—Å–∫–æ–π –∑–∞–ø–∏—Å–∏ (`_key_out`)**.
    *   –í—ã—á–∏—Å–ª—è–µ—Ç **–∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç —Å—Ö–æ–¥—Å—Ç–≤–∞ (`_score`)** —Å –∫–∞–Ω–æ–Ω–∏—á–µ—Å–∫–æ–π –∑–∞–ø–∏—Å—å—é.
- **–ü—Ä–∏–º–µ—Ä –∏–∑ –∫–Ω–∏–≥–∏:** –ü–æ–∏—Å–∫ –¥—É–±–ª–∏–∫–∞—Ç–æ–≤ –∫–ª–∏–µ–Ω—Ç–æ–≤ –≤ —Å–ø–∏—Å–∫–µ.
    *   **–í—Ö–æ–¥:** "John Smith, 123 Main St.", "Jon Smith, 123 Main Street", "J. Smith, 123 Main St.".
    *   **–†–µ–∑—É–ª—å—Ç–∞—Ç Fuzzy Grouping:** –í—Å–µ —Ç—Ä–∏ –∑–∞–ø–∏—Å–∏ –ø–æ–ª—É—á–∞—Ç –æ–¥–∏–Ω `_key_out` (–∫–ª—é—á –∫–∞–Ω–æ–Ω–∏—á–µ—Å–∫–æ–π –∑–∞–ø–∏—Å–∏, –Ω–∞–ø—Ä–∏–º–µ—Ä, –¥–ª—è "John Smith, 123 Main St.") —Å –≤—ã—Å–æ–∫–∏–º–∏ `_score` (0.95, 0.92). –≠—Ç–æ —Å–∏–≥–Ω–∞–ª, —á—Ç–æ —ç—Ç–æ –æ–¥–∏–Ω –∏ —Ç–æ—Ç –∂–µ –∫–ª–∏–µ–Ω—Ç.
- **–ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ:** –ü–æ—Å–ª–µ –≥—Ä—É–ø–ø–∏—Ä–æ–≤–∫–∏ –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å Conditional Split, —á—Ç–æ–±—ã –æ—Å—Ç–∞–≤–∏—Ç—å —Ç–æ–ª—å–∫–æ –∫–∞–Ω–æ–Ω–∏—á–µ—Å–∫–∏–µ –∑–∞–ø–∏—Å–∏ (`_key_in == _key_out`) –∏–ª–∏ –∞–≥—Ä–µ–≥–∏—Ä–æ–≤–∞—Ç—å –¥–∞–Ω–Ω—ã–µ –ø–æ –≥—Ä—É–ø–ø–∞–º.

### **3. –û—á–∏—Å—Ç–∫–∞ —á–µ—Ä–µ–∑ Data Quality Services (DQS)**

- **Data Quality Services (DQS)** ‚Äî —ç—Ç–æ **–æ—Ç–¥–µ–ª—å–Ω—ã–π —Å–µ—Ä–≤–µ—Ä–Ω—ã–π –∫–æ–º–ø–æ–Ω–µ–Ω—Ç SQL Server**, –ø—Ä–µ–¥–Ω–∞–∑–Ω–∞—á–µ–Ω–Ω—ã–π –¥–ª—è —É–ø—Ä–∞–≤–ª–µ–Ω–∏—è –∫–∞—á–µ—Å—Ç–≤–æ–º –¥–∞–Ω–Ω—ã—Ö.
